# Fake News Classifier Project
# Беликов, Баранова, Орехов, Лешуков. Группа 423


## Описание разработанной системы

В лабораторной работе была реализована модель-классификатор новостей на реальные и сгенерированные. Для векторизации текта был использован Word2Vec, преобразующий слова в эмбеддинги (векторные представления). Для улучшение качества обучения модели была применена аугментация для увеличения выборки. После аугментации появилась необходимость в регулировке весов для разных классов, потому как аугментация применялась только для сгенерированных новостей. В процессе обучения были собраны метрики и составлены графики.

### Архитектура
- Система парсинга новостей с новостного сайта (В частности apnews)
- Система генерации фейковых новостей (на основе локальных GPT-2 и GPT-NEO)
- Система подготовки датасета
- Система обучения и валидации модели

### Алгоритмы работы разработанной системы
1. Парсинг новостей с сайта и последующая обработка полученного текста (удаление свойственных кокретным сайтам конструкций и меток, приведение оформления текстов к единому, более пригодному к обучению, формату)
2. Генерация фейковых новостей на основе локальных версий моделей GPT-2 и GPT-NEO. Обработка полученных результатов и приведение к единому формату
3. Аугментация данных (сгенерированных) и подготовка датасета к применению в обучении модели-классификатора.
4. Сбор, анализ и представление метрик
